# ⚔ 単語ウォーズ

英単語学習を「対戦ゲーム」にした無料のWebアプリです。インストール不要、ブラウザで動きます。

**▶ 遊ぶ: https://noharag.github.io/tango-wars/**

スマホなら、開いた後にブラウザのメニューから「ホーム画面に追加」するとアプリとして使えます(オフライン対応)。

## どんなゲーム?

- **⚔ ランク対局** — 1分間、ライバル(ボット)とスコア勝負。勝てば達成率が上がり、30級→初段→…→九段へ昇級
- **🔥 フィーバーチェイン** — 連続正解10ごとにフィーバー。フィーバー中に繋げばLv4(スコア×5)まで過熱、BGMも転調
- **⚡ ブリッツ60** — 1問3秒×60秒の自己ベスト勝負
- **📱 ワードフィード** — ショート動画式の無限スクロールで単語を浴びる
- **📖 図鑑** — 覚えた単語をレア度付きで捕獲・コレクション
- **🎰 スカウトガチャ / デイリークエスト / ログインボーナス / 週次シーズン**

裏側では忘却曲線(SRS)が回っていて、**遊んでいるだけで復習が最適なタイミングで巡ってきます。**

## 収録単語

全2287語、すべて例文・コロケーション付き:

- 一般英単語(中学基礎〜英検1級レベル) 1071語
- 大学院入試・学術論文の頻出語 538語
- IT用語 307語 / ロボット工学用語 371語(エンジニア向け)

## データについて

- 学習データは**あなたのブラウザ(localStorage)にのみ**保存されます。サーバーには何も送信されず、利用者ごとに完全に独立しています
- 端末を変えるときは 設定 → エクスポート/インポート で引っ越しできます
- ブラウザのサイトデータを消すと学習履歴も消えます(定期的なエクスポート推奨)

## ローカルで動かす

```
git clone https://github.com/noharaG/tango-wars.git
cd tango-wars
python -m http.server 8613   # または server.bat (Windows)
```

`index.html` を直接開いても動きます(PWA機能のみ無効)。ビルド不要・依存なしの素のHTML/CSS/JSです。

## 注意

- 個人の学習用に作った趣味プロジェクトです。単語データはAI生成のため、まれに誤りが含まれる可能性があります(Issueで教えてもらえると助かります)
- 効果音・BGMはWebAudioでリアルタイム合成しており、外部素材は使用していません
