# 単語ウォーズ (TANGO WARS) — ゲームデザイン書 v1.0

自分専用の英単語学習アプリ。目的は「英語がペラペラになる」こと。
v1 = 英単語中心。v2 (Season 2) = 英作文中心(瞬間英作文・並べ替え)。データは最初から英作文対応(例文・コロケーション収録)。

## 0. ユーザープロファイル

- ロボットエンジニア。英語はほぼ話せない。語彙は初学者〜上級まで穴がある前提で全レベル収録。
- 専門語彙(大学院入試レベルの学術語・IT用語・ロボット工学用語)を重点収録。
- 自己申告「ドパガキ」= ドーパミン駆動型。即時報酬・可変報酬・短周期ループに最適化する。

## 1. 中毒設計の根拠 — 過去にハマったゲームの共通項

| ゲーム | 抽出した中毒メカニクス | 本アプリへの実装 |
|---|---|---|
| 将棋ウォーズ | 3〜10分の対戦、段級位、達成率ゲージ、「あと1局」 | **1分ランク対局**(ボット対戦)、30級→九段、達成率0-100% |
| 妖怪ウォッチぷにぷに | コンボ連鎖→フィーバータイムの爆発 | 連続正解コンボ→**フィーバー**(15秒スコア2倍+演出激変) |
| モンスト | ガチャ(可変報酬)、短周回、キャラ収集 | コインで**単語スカウトガチャ**、対局=数分周回、**図鑑** |
| 三国志真戦 | シーズン制、長期の積み上げ | **週次シーズン**、過去の自分とのゴースト比較 |
| 人狼ジャッジメント | 読み合い・ひっかけを見破る快感 | 誤答選択肢を**意図的なひっかけ**(類義語・似た綴り)に設計 |
| 囲碁 | レート=上達の数値化 | 内部Eloレート+推移グラフ |

設計原則(ドパガキ最適化):
1. **1ループ1分強**。結果(勝敗・昇級ゲージ変動)は必ず数値と演出で返す。
2. **可変報酬**: 正解時にランダムでボーナスコイン(確率10%で×3)。ガチャ。捕獲演出。
3. **即時フィードバック**: 正解→0.1秒以内に音+パーティクル+スコアポップ。待たせない。
4. **損失回避**: ストリーク(連続日数)とコンボは「切らしたくない」設計。ただし罰は軽く(学習が嫌にならない)。
5. **学習とゲームを分離しない**: ガチャの排出物は「単語」自体。遊ぶ=学ぶが常に一致。

## 2. コアループ

```
ホーム → [対局開始] → 1分バトル(出題ループ) → リザルト(勝敗/達成率/報酬/捕獲)
  ↑                                                        ↓
  ← ガチャ/図鑑/クエスト受取 ←—— コイン・新スカウト単語 ——←
```

### 2.1 ランク対局 (メインモード)

- 制限時間 **60秒**(180秒→60秒→30秒と試して60秒で確定)。プレイヤー vs ボット(ライバル)のスコア勝負。
- ボットは自分のEloに応じた正答率・回答速度で自動的にスコアを刻む(画面上部に両者のスコアバー)。
- 出題は4択(+PC設定でタイピング)。正解でスコア、誤答でコンボ切れ。
- 終了時スコアが高い方が勝ち。勝ち→達成率+25%前後、負け→-15%前後。100%で**昇級演出**。
- 30級から始まり 1級 → 初段 → … → 九段。

### 2.2 特訓モード (フリー対局)

- レート変動なし。SRSの復習期限が来た単語を集中消化。時間無制限、10問1セット。
- 「今日の復習 ○語」バッジでホームから直行。

### 2.3 出題の裏側 = SRS (忘却曲線)

- 全単語に SM-2 簡略版の状態を持たせる。対局中の正誤・回答速度がそのまま SRS に反映。
- 出題キュー: 復習期限到来 60% / 新規 30% / ランダム復習 10%。
- **プレイヤーは SRS を意識しない**。ゲームを遊ぶと勝手に忘却曲線が回る。

## 3. 収集・育成

- 単語レア度 = レベル帯: N(中学) / R(高校) / SR(大学受験) / SSR(上級・英検1級/学術) / UR(IT・ロボット専門)。
- mastery 3 以上で「**捕獲**」= 図鑑登録。mastery 5 + interval 21日以上で「**キラ化**」。
- **スカウトガチャ**: 100コインで未習得語から3語をレア度抽選でスカウト → 次の対局に優先出題。スカウト語の捕獲はコイン2倍+専用演出。
- 実績(称号): 初昇級、コンボ20、UR捕獲、ストリーク7日 など → ホームのフレーム解放。

## 4. 継続装置

- **デイリークエスト** 3件(日付シードで決定的に選出): 対局1回/復習20語/新出5語/コンボ15/勝利1回 など → コイン。
- **ストリーク**: 1日1対局(or 20問)で継続。コイン獲得倍率 +5%/日(上限+50%)。
- **週次シーズン**(月曜0時リセット): 週間スコア累計。過去の自分の週間ベストがゴーストとして表示され、超えるとシーズン報酬。

## 5. ロードマップ

- **v1 (今回)**: 上記全部。英作文モードはメニューに「Season 2 で解禁」とロック表示(予告=モチベ)。
- **v2**: 瞬間英作文(日本語→単語チップ並べ替え→タイプ入力の段階制)。収録済みの例文・コロケーションをそのまま使う。
- 端末間同期は v1 ではデータのエクスポート/インポート(JSON)で代用。

## 6. 技術方針

- 素の HTML/CSS/JS(フレームワークなし、ビルド不要)。`window.TW` 名前空間、script タグ順次読み込み。file:// でも動く。
- データ永続化: localStorage。単語データは `data/words.js`(JS ファイルなので fetch 不要)。
- PWA(manifest + Service Worker)は http(s) 配信時に有効。ローカルは `server.bat`(python http.server)。
- スマホ用に全ファイルを1枚に固めた `dist/tango-wars.html` を `tools/build.py` で生成。
- 効果音は WebAudio でシンセ生成(外部アセットなし・オフライン完結)。発音は speechSynthesis(端末内蔵TTS)。
